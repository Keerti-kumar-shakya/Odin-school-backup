Grid Introduction

1.) What is Grid?

Ans : To Understand the Key Differences Grid utilizes a two-dimensional system that allows placing items in rows and columns, making it easier to create complex layouts. To get started you have to define a container element as a grid with display: grid , set the column and row sizes with grid-template-columns and grid-template-rows , and then place its child elements into the grid with grid-column and grid-row .

2.) Why is grid used?

A grid can be used to organize graphic elements in relation to a page, in relation to other graphic elements on the page, or relation to other parts of the same graphic element or shape.
