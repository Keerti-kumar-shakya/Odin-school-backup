display property value below

  • block: new line, complete width it will take, can apply width & height
  • inline: no new line, will take only content size it will not take complete width, cannot apply width & height
  • to your default inline element if you want to give width & height, for that update display property to inline-block inline-block: it will act as inline element but we can apply width & height
  • none: element will not appear in screen;
  • flex Flex-container : an element which help to define the layout of its children Flex-item:- all children element of a flex container
  • grid
  • table
  • list-item
